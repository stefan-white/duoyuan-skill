## 三大业务板块

### 板块一：多元智能（核心）

**定位**：汽车智能产线解决方案提供商 / 汽车智能产线创领者

提供从概念设计到设备研发、数字化产线部署、智慧运维的 EPC 交钥匙工程服务。

**核心产品线**：

#### 焊装生产线
白车身焊装线、车身下车体线、车身主焊线、空中转接线、分总成线、焊接机器人工作站、单轴/双轴变位机、焊接夹具

#### 总装生产线
车身总装线、底盘装配线、内饰装配线、车门分装线、轮胎分装线、检测线

#### 涂装生产线
涂装线整体解决方案

#### 自动化输送设备
地板链输送线（单/双）、EMS 空中输送线、WBS 存储输送线、PBS 存储输送线、高位悬挂线、辊床+滑撬输送系统、AGV 智能物流输送

#### 智能物流仓储
重载 AGV、MALS 柔性输送系统（自主研发）

#### 机器人应用
焊接机器人、点焊机器人、搬运机器人

#### 其他装备
辊压设备、开卷机、动力网架、张拉设备、车身骨架合装设备、顶蒙皮自动点焊设备、助力机械手臂

**MALS 系统**（自主研发核心技术）：
- 以智能 AGV 实现柔性输送
- 无需地面基础，可根据工艺快速重构生产线
- 与 MES 对接，实现移动装配、按需移动和管理
- 结构形式：上潜式、牵引式、背负移载式、定制装配式、物流叉车式

**解决方案**：
- 智能产线解决方案（焊装 + 总装 + 涂装）
- 工业 4.0 智能工厂解决方案
- 数字孪生体系
- EPC 交钥匙工程（概念设计 → 设备研发 → 数字化产线部署 → 智慧运维）
