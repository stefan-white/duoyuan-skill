# 多元智能 AI Skill

> 郑州多元智能装备股份有限公司 · 汽车智能产线创领者

为 AI 助手提供多元智能的公司信息、产品线、项目案例和解决方案查询能力。

## 安装

将 `duoyuan-skill/` 目录复制到对应平台的 skills 目录：

| 平台 | 目录 |
|------|------|
| Claude Code | `.claude/skills/duoyuan-skill/` |
| Cursor | `.cursor/skills/duoyuan-skill/` |
| Windsurf | `.windsurf/skills/duoyuan-skill/` |
| Trae | `.trae/skills/duoyuan-skill/` |
| 通用 | `.agents/skills/duoyuan-skill/` |

## 能力覆盖

- 公司基本信息与发展历程
- 三大业务板块介绍（多元智能 / 多元车身 / 多元注塑）
- 产品线详情（焊装线、总装线、涂装线、输送设备、AGV、机器人等）
- 解决方案（智能产线 / 工业 4.0 / EPC 交钥匙工程 / 数字孪生）
- 典型项目案例（国内 + 海外）
- 资质荣誉（专精特新小巨人、高新技术企业等）
- 联系方式（中英双语）

```bash

```
